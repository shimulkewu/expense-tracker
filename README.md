# 💰 Smart Expense Tracker (Python CLI)

A simple Python-based expense tracker that allows users to record, view, and analyze daily expenses.

---

## 🚀 Features

- Add new expenses
- View all expenses
- Calculate total spending
- Save data in CSV file
- Simple CLI interface

---

## 🛠️ Tech Stack

- Python 3
- CSV module
- (Optional) Pandas & Matplotlib

---

## ▶️ How to Run

```bash
python3 main.py
